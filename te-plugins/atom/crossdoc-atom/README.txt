##README for CrossDoc Atom Plugin


NOTE: Until final build phase, not implemented through Atom's Paackage Manager. Currently must install through manual insertion of files.

ENSURE YOU HAVE:
-ATOM, atom-runner package,
edit ~/.atom/config.cson

with

	runner:
		scopes:
			js:"node"
			
			


-Insert crossdoc-atom to sub-directory: Drive:/Users/(Username)/.atom/packages
-reload Atom window, navigate to Packages where CrossDoc should be available.
